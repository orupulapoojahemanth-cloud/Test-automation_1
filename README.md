Playwright TypeScript Automation Framework
About

This project is a UI automation framework built using Playwright and TypeScript. It follows the Page Object Model (POM) design pattern and uses Playwright hooks to organize reusable test setup.

Features:
1)Page Object Model (POM)
2)Playwright Hooks (beforeEach)
3)Login automation scenarios
4)Positive and negative login validation
5)Playwright assertions and locators
6)Cross-browser execution support
7)Tech Stack
8)Playwright
9)TypeScript
10)Node.js
11)Visual Studio Code


Installation:
npm install
Run Tests
npx playwright test

Run in headed mode:

npx playwright test --headed
